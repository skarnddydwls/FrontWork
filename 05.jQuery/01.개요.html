jQuery 개요

* jQuery란

- 기존의 자바스크립트 언어 만으로 구현하기 복잡했던 구문들을 간소화하기 위해 '존 레식'에 의해 만들어짐
- javaScript 기반 라이브러리
 즉, javaScript언어로 유용한 함수나 내용들을 이미 정의되어 있는 모음집
- 복잡한 자바스크립트 구문보다 짧고 쉽게 개발 가능

** jQuery의 장점
1) DOM요소와 관련된 스크립트를 보다 쉽게 구현할 수 있음
2) Ajax(비동기식 통식), 이벤트 처리 등등 폭 넓게 지원함
3) jQuery와 관련된 확장형 플러그인, 오픈 api등을 지원
 (차트, 슬라이드 등등...)

- head태그 내에 script 태그로 외부 jQuery 관련 ???.js 파일 연결하기

* jQuery 작성 방법

jQuery 구문으로 DOM 요소를 다루는 구문을 기본적으로 작성하기 때문에
문서상에 요소들이 다 만들어지고나서 실행되어야 함

- 순수 자바 스크립트와 jQuery비교 

    window.onload
    window.onload = function() {
        실행내용; 
    }

    해당 문서의 요소들이 모두 로드되고 뿐만 아니라
    연동되어 있는 외부 문서들도 모두 로드 된 후에 실행됨
    그리고 여러번 쓰면 맨 마지막것으로 덮어쓰기

    즉시 실행 함수 ilfe
    ([변수] = function([매개변수, ...]) {
        실행내용;
    }) (인자, ...);

    브라우저가 해당 줄의 코드를 읽는 순간 즉시 실행
    즉, html이나 이미지 같은 외부 리소스의 로드와 상관 없이 실행
    자바스크립트 파일이 파싱되는 중간에 곧바로 실행
    그리고 몇 번이든 사용 가능

    jQuery
    자바스크립트 파일이 파싱이 모두 되어 DOM트리만 완성되면 실행
    html이나 이미지 같은 외부 리소스의 로드와 상관없이 실행
    그리고 몇 번이든 사용 가능

- jQuery 구문 
    1) jQuery(document).ready(function() {
            실행내용;
    });
    2) $(document).ready(function(){
            실행내용;
    });
    3) $(function() {
            실행내용;
    });
    4) $(() => { 
            실행내용;
    });
